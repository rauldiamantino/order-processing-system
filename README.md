# Order Processing System

## 📌 Overview
This project simulates an async order processing system, inspired by real-world e-commerce scenarios.

## 🚀 Tech Stack
- Java 17
- Spring Boot
- REST APIs

## ⚙️ Features
- Create order
- Retrieve order
- Basic status simulation

## 🧠 Context
Simulates real-world problems such as:
- Order lifecycle
- External integrations (ERP simulation)
- Failure scenarios (to be implemented)

## 🔥 Next Steps
- Database integration
- Async processing with queue
- Retry strategy
